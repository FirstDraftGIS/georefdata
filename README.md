# georefdata
Country Codes and other Miscellaneous Geo-Spatial Reference Data

```
import georefdata

georefdata.get_country_codes()
# ['AF', 'AL', 'DZ', 'AS', 'AD', ... 'ZW']

georefdata.get_geonames_feature_classes()
# ['A', 'H', 'L', 'P', 'R', 'S', 'T', 'U', 'V', 'n']

georefdata.get_geonames_feature_codes()
# ['ADM1', 'ADM1H', 'ADM2', ... 'ZNF', 'ZOO', 'null]
```
